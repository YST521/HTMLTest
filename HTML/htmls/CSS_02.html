流式布局

流式布局: 本质上是百分比布局 (百分比是基于父级的大小) (外接 .css文件)

响应式布局 : 根据浏览器宽度不同使用不同的布局风格(根据缩放浏览器窗口 内容进行变化) (外接 .css文件)


    大于 1000px " ( link media="screen and (min-width:1000px)" rel="stylesheet" type="text/css" href="css/base03.css"/>> "

小于 600px ( link media="screen and (max-width:600px)" rel="stylesheet" type="text/css" href="css/base01.css"/>

大于 600px 小于 1000px ( link media="screen and (min-width: 600px) and (max-width: 1000px)" rel="stylesheet" type="text/css" href="css/base02.css"/>

2、亚马逊中国停止第三方商家服务

4月18日中午，根据新浪科技的报道，亚马逊中国确认，将于2019年7月18日停止为亚马逊中国网站上的第三方卖家提供卖家服务

亚马逊中国称，在过去几年中，亚马逊中国持续聚焦并发力跨境在线零售业务，获得了中国消费者的积极反馈和认可，为了深化这一战略转型，我们将于2019年7月18日停止为亚马逊中国网站上的第三方卖家提供卖家服务。此外，将与所有卖家紧密合作，完成后续交接事宜，以确保持续为用户提供优质的购物体验。同时，此次调整中受到影响的卖家如希望继续与亚马逊合作并拓展全球市场，可以联络亚马逊全球开店团队获得帮助。

3、网曝三星Galaxy Fold屏幕出现问题

北京时间4月18日下午消息，据美国财经网站CNBC报道，三星电子股价今日下跌逾3%，原因是在其首款可折叠屏幕手机Galaxy Fold即将正式开售之前，媒体评测人员却给出了不好的消息。Galaxy Fold在今年的世界移动通信大会前夕发布，售价1980美元，目前仅提供给媒体供评测人员使用，预计于本月晚些时候在美国和其他一些国家及地区正式开售。

对Galaxy Fold在评测中出现的各种屏幕问题，三星在一份声明中称，正在评估这些问题报告。三星称：“我们向媒体提供了数量有限的早期版本的Galaxy Fold样机，供其评测。近日，我们收到了有关屏幕问题的评测报告，公司将对这些问题手机展开彻底调查，以确定原因。”
